## Nestjs and GraphQl

## Description

A simple Graphql backend API using Nestjs framework and mongodb;

## Installation

```bash
$ npm install

[Open localhost:3000/graphql](http://localhost:3000/graphql)
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
